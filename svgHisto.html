svgHisto

Structure de l'objet svgHisto

Code:

var myHisto2=new svgHisto_Model({   
    id : "histoBar2",
    type: "bottomup",
    x: 100,
    y: 50,
    w: 200,
    h: 300,
    interBar: 5,
    data: mdata,
    labelKey: "dateExtraction",
    valueKey: "Effectif_reel",
    svgBar_Style:{  
        id : "testBar", 
        type: "jsBarTopBottom",
        x: 10,
        y: 50,
        size: 100,
        thickness: 50,
        svgBar_label:  {
            "text":"Label",
            "dx": 0,
            "dy": 0
            },
        svgBar_label_position: "centerMiddle",
        svgBar_rect:    {   
            "style":"cssdrawZone1"
            },
        svgBar_value:  {
            text: 10,
            "dx": 0,
            "dy": -5
            },
        svgBar_value_position: "topMiddle",
        });

Description:

ParamétreValeur par défautDescription
id : nullid de l'objet histogramme
type : nulltype d'histogramme topDown, bottomUp, leftRight, rightLeft
x : 0position horizontale de l'histogramme
y : 0position verticale de l'histogramme
w : 0largeur de l'histogramme
h : 0hauteur de l'histogramme
interBar :5espace entre les bars
data : 0table de donnée
labelKey :0clé de la table contenant les titres
valueKey :0clé de la table contenant les valeurs
svgBar_Style :0svgBar objet ou structure JSON de svgBar

Defaut css style:

aucun

Code:

var affichenBar=new svgBar_ModelView({el: $("#Abar"), model: nBar });

Description:

ParametreValeur par défautDescription
el :nullsvg élément, où dessiner l'histogramme
model:nullobjet model
