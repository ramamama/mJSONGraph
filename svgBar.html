svgBar

Structure de l'objet svgBar

Code:

var nBar=new svgBar_Model({ 
	id : "testBar", 
	type: "jsBarTopBottom",
	x: 10,
	y: 50,
	size: 100,
	thickness: 50,
	svgBar_label: {
		"text":"Label",
		"dx": 0,
		"dy": 0
		},
	svgBar_label_position: "centerMiddle",
	svgBar_rect: {   
		"style":"cssdrawZone1"
		},
	svgBar_value:  {
		text: 10,
		"dx": 0,
		"dy": -5
		},
	svgBar_value_position: "topMiddle",
	});

Description:

ParametreValeur par défautDescription
id : nullid de l'objet Bar doit être définit ou est transmis par l'objet appelant
type : ""jsBarLeftRight ou jsBarTopBottom
x : 0position horizontale de la bar
y : 0position verticale de la bar
size : 0Taille de la Bar
thickness : 0Epaisseur de la Bar
svgBar_rect : 0svgRect objet ou structure JSON de svgRect
svgBar_label : 0svgText objet ou structure JSON de svgText
svgBar_label_position : 0cf. position des textes par rapport à la forme centrale
svgBar_value :0 svgText objet ou structure JSON de svgText
svgBar_value_position : 0cf. position des textes par rapport à la forme centrale

Position des textes par rapport à la forme centrale:

Exemples de bar avec les différentes positions de textes

Defaut css style:

none

Rendu de l'objet

Code:

var affichenBar=new svgBar_ModelView({el: $("#Abar"), model: nBar });

Description:

Parametre Valeur par défautDescription
el : nullsvg élément, où dessiner la bar
model: nullobjet model
