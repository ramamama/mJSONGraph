svgRect

Structure de l'objet svgRect

Code:

var monRect=new svgRect_Model({
    id : "monRect1",
    x: 10,
    y: 10,
    w: 80,
    h: 80,
    rx: 0,
    ry: 0,	
    style:"svgRectDefault"
    });

Description:

ParametreValeur par défautDescription
id : nullid de l'objet svgRect doit être définit ou est transmis par l'objet appelant
x : 0position horizontale du rectangle
y : 0position verticale du rectangle
w : 0largeur du rectangle
h : 0hauteur du rectangle
rx : 0attribut x d'arrondissement du rectangle
ry : 0attribut y d'arrondissement du rectangle
style : "svgRectDefault"style par défaut du rectangle

Defaut css style:

.svgRectDefault {fill: red; }

Rendu de l'objet

Code:

var afficheMonRect=new svgRect_ModelView({el: $("#RectE"), model: monRect });

Description:

Parametre Valeur par défautDescription
el : nullsvg élément, où dessiner le rectangle
model: nullobjet model

Exemple:


//on creer le canvas
