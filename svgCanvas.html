svgCanvas

Structure de l'objet svgCanvas

Code:

var mCanvas=new svgCanvas_Model({
	id:"monCanvas", 
	x: 0, 
	y: 0, 
	w: 200, 
	h: 200, 
	style:"mCanvas"
	});

Description:

ParametreValeur par défautDescription
id : nullid de l'objet Canvas doit être définit ou est transmis par l'objet appelant
x : 0position horizontale du canvas
y : 0position verticale du canvas
w : 0largeur du canvas
h : 0hauteur du canvas
style :"svgCanvasDefault"style par défaut du canvas

Defaut css style:

.svgCanvasDefault {background-color: grey}

Rendu de l'objet svgText

Code:

var maVue=new svgCanvas_ModelView({el:"#mleft ", model: mCanvas});

Description:

Parametre Valeur par défautDescription
el : nullsvg élément, où dessiner le texte
model: nullobjet model

Exemple:

var mCanvas=new svgCanvas_Model({
	id:"monCanvas", 
	x: 0, 
	y: 0, 
	w: 200, 
	h: 200, 
	style:"mCanvas"
	});
var maVue=new svgCanvas_ModelView({el:"#mleft ", model: mCanvas});
