svgText

Structure de l'objet svgText

Code:

var untext=new svgText_Model({  
	id : "texte1",
	text:"Exemple de texte",
	x: 10,
	y: 10,
	dx: 0,
	dy: 0,
	transform:"",
	style:"svgTextDefault"
	});

Description:

ParametreValeur par défautDescription
id : nullid de l'objet Text doit être définit ou est transmis par l'objet appelant
text: ""texte qui sera affiché
x : 0position horizontale du texte
y : 0position verticale du texte
dx : 0décalage horizontal du texte
dy : 0décalage verticale du texte
style :"svgTextDefault"style par défaut du texte

Defaut css style:

.svgTextDefault {font-family: Arial; font-size: 34; text-anchor: middle;}

Rendu de l'objet svgText

Code:

var afficheTxt=new svgText_ModelView({el: mtext, model: untext });

Description:

Parametre Valeur par défautDescription
el : nullsvg élément, où dessiner le texte
model: nullobjet model

Exemple:

//on creer le code svg et on lui affecte id=mText1
d3.select("#mleft").append("svg").attr("id","mText1");

	dx: 0,
	dy: 0,
	transform:"",
	style:"ligne3"
    });
//on affiche l'objet untext à l'emplacement mText1
